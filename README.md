## Tech Stack 🛠️

- **Backend:** Node.js, Express.js, Mongoose  
- **Database:** MongoDB, 
- **Authentication:** JWT  

## Installation ⚙️

Clone the repo and install dependencies:

```bash
git clone https://github.com/itamim-coder/everything-green-task.git
cd everything-green-task
npm install
```

Run the application:

```bash
nodemon start-dev
```
